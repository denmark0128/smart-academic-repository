Project Description :

The purpose of the study was to develop Patient Monitoring Application for Pasig City Children�s Hospital. The application was developed to secure patient information and so the nurses can easily manage patient record in most efficient way their record will no longer on need to be filled in hard copies that occupies space and time. The developer created a database system that will help the reduction of data redundancy to help the management in organizing the records. Report generation can be easily done and would not take a lot of time because everything is kept in a single database. The developer included user level of access for Administrator are Add Patient, Set Medication Time, Transfer, Discharge and Generate report while for the User are receiving the Notification of Medication of Patient, Recording the Vital Sign and Updating the Vital Sign. The developer assured the system is accessible, accurate, user friendly, reliable and convenient to assist the users.
